OPTION 1: RENDER.COM (⭐ Recommended - Easiest & Fastest)
──────────────────────────────────────────────────────────────────────

Why Render?
• render.yaml already configured in your repo
• Free SSL certificate
• Auto-deploy on git push
• Free PostgreSQL database
• 5-minute setup

Steps:
1. Go to: https://dashboard.render.com
2. Sign up/Login with GitHub
3. Click "New +" → "Web Service"
4. Select repo: KINGSACCOUNT1/coach-jv
5. Render auto-detects everything ✨
6. Add environment variables:
   - SECRET_KEY (generate one)
   - ZOHO_EMAIL=support@coachjvtech.us
   - ZOHO_APP_PASSWORD=<your-password>
7. Click "Create Web Service"
8. Wait 5-10 minutes ☕
9. Get your URL: coach-jv-xyz.onrender.com

Then update Cloudflare:
• Go to Cloudflare DNS
• Change @ record to: CNAME → coach-jv-xyz.onrender.com
• Change www record to: CNAME → coach-jv-xyz.onrender.com
• Set proxy: ON (orange cloud) ☁️

Done! 🎉
